# MRPZ_Lab_1 — Сума елементів матриці відносно головної діагоналі
 
## Завдання
 
Дано матрицю A розміром M×N, де M=4, N=4. Обчислити суми елементів матриці, розташованих **вище** і **нижче** головної діагоналі.
 
## Математичний сенс
 
**Головна діагональ** — елементи, де номер рядка дорівнює номеру стовпця (`i == j`).
 
**Вище діагоналі** — всі елементи правіше від діагоналі (`j > i`).
 
**Нижче діагоналі** — всі елементи лівіше від діагоналі (`j < i`).
